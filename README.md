KissanConnect - Professional Static Landing Page (ready for GitHub Pages)

How to publish:
1. Create a new GitHub repository named `kissanconnect`.
2. Upload the contents of this ZIP to the repository root (or push via git).
3. In the GitHub repo: Settings → Pages → Branch: main, Folder: / (root) → Save.
4. After a minute, your site will be live at:
   https://<your-username>.github.io/kissanconnect/

Files included:
- index.html
- styles.css
- script.js
- assets/ (logo.svg, hero.svg, crops/*.svg, icons/*.svg)

Notes:
- All asset paths are relative and GitHub Pages compatible (./assets/...).
- Contact form is UI-only. To receive messages, connect a backend or use a form service (e.g., Formspree).
